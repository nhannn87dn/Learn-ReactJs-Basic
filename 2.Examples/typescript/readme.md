# Học TypeScript cơ bản

- Cài môi trường: NodeJS <https://nodejs.org/en/download/>

- Tạo dự án TypeScript

Step 1: npm init -y
Step 2: npx tsc --init
Step 3: npm install -g typescript

- Chạy một file typescript .ts  

```bash 
tsc index.ts
```
Biên dịch ra file index.js

```bash 
node index.js
```

# ⭐ Thêm Css vào Dự Án

## 🔥 0. CSS Module ?

* **CSS Modules là gì?** Giải thích ngắn gọn về khái niệm (là các tệp CSS mà trong đó tất cả tên lớp và tên hoạt ảnh được định phạm vi cục bộ - local scope - theo mặc định).
* **Tại sao nên dùng?**
* Tránh xung đột tên class giữa các component (Global Scope vs Local Scope).
* Dễ dàng quản lý, bảo trì và xóa bỏ khi component không còn sử dụng.
* Không cần cấu hình phức tạp (hỗ trợ sẵn trong Vite, Create React App, Next.js).

---

### Cấu trúc Thư mục Dự án (Best Practice)

Gợi ý cách tổ chức thư mục để đặt các tệp CSS Module một cách khoa học:

```text
src/
├── components/
│   ├── Button/
│   │   ├── Button.jsx          # Component React
│   │   └── Button.module.css   # CSS Module cho Button
│   └── Card/
│       ├── Card.jsx
│       └── Card.module.css
├── App.jsx
└── main.jsx

```

> **Lưu ý:** Quy ước đặt tên bắt buộc phải có đuôi `.module.css` (hoặc `.module.scss` nếu dùng SASS).

### Các bước Triển khai Chi tiết

#### Bước 1: Tạo file CSS Module

Viết CSS như bình thường, sử dụng các tên class ngắn gọn, tường minh.

```css
/* Button.module.css */
.btn {
  padding: 10px 20px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

.primary {
  background-color: #007bff;
  color: white;
}

.secondary {
  background-color: #6c757d;
  color: white;
}

```

#### Bước 2: Import và Sử dụng trong Component React

Import file CSS dưới dạng một đối tượng (object) và truyền property vào `className`.

```jsx
// Button.jsx
import React from 'react';
import styles from './Button.module.css'; // Import object 'styles'

const Button = () => {
  return (
    {/* Kết hợp nhiều class bằng Template Literals */}
    <button className={`${styles.btn} ${styles.primary}`}>
      Button Default
    </button>
  );
};

export default Button;

```

## 🔥 3. Sass (Advanced) ?

SASS/SCSS là một chương trình tiền xử lý CSS (CSS preprocessor). Nó giúp bạn viết CSS theo cách của một ngôn ngữ lập trình, có cấu trúc rõ ràng, rành mạch.
